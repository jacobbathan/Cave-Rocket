# Cave Rocket
## A 2D rocket game built in Unity 3D.


Demo avaliable [at this link.](https://simmer.io/@jacobbathan/caverocket)


### Controls
| Key      | Description |
| ----------- | ----------- |
| A      | Rotate Left       |
| D   | Rotate Right        |
| Space   | Throttle        |

Success chime made in FamiStudio. 
Background track made in Garageband.
